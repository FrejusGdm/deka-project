## Quick Start

Get started with Deka in just a few lines of code:

<CodeGroup>

```python Basic Translation
import deka

# Configure with your API keys
deka.configure({
    'openai_api_key': 'your-openai-key',
    'ghananlp_api_key': 'your-ghananlp-key'
})

# Simple translation
result = deka.translate("Hello world", "french")
print(result.text)  # "Bonjour le monde"
```

```python Model Selection
import deka

# Choose specific AI models
result = deka.translate(
    "Hello world", 
    "spanish", 
    provider="openai/gpt-4"
)
print(result.text)  # "Hola mundo"
print(result.metadata['model'])  # "gpt-4"
```

```python African Languages
import deka

# Translate to African languages
result = deka.translate(
    "Thank you", 
    "twi",  # Ghanaian language
    provider="ghananlp"
)
print(result.text)  # "Meda wo ase"
```

```python Provider Comparison
import deka

# Compare multiple providers
comparison = deka.compare("Hello", "spanish", providers=[
    "google",
    "openai/gpt-4", 
    "anthropic/claude-3-5-sonnet"
])

for result in comparison.results:
    print(f"{result.provider}: {result.text}")
```

</CodeGroup>
